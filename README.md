# Maître Chocolatier - Landing Page

Landing page élégante pour un chocolatier artisanal français.

## Design

- **Style**: Éditorial / Luxe / Minimaliste
- **Couleurs**: Or (#c9a962), Noir profond (#1a1a1a), Blanc cassé (#faf8f5)
- **Typographie**: Geist Sans + Playfair Display

## Sections

- Navigation fixe
- Hero avec CTA
- Notre Maison (histoire)
- Collections de produits
- Savoir-faire (4 étapes)
- Témoignages clients
- Contact + formulaire
- Footer

## Déploiement

Le site est déployé automatiquement via Vercel.
